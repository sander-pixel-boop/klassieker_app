## 2026-04-05 - Consistent Placeholder Guidance
**Learning:** Text inputs across different pages offering the same functionality (e.g. database search) should have consistent placeholders. In this app, Scorito had a helpful 'bijv. Van Aert of Visma' placeholder, but the Sporza page did not.
**Action:** When adding text inputs for searches, always include a realistic example in the placeholder, and ensure parity between similar tools.
